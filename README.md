# Klein Quiz 1.0 – The Computer Technician Challenge

Klein Quiz 1.0 is a free educational quiz developed by Klein Dream.

Test your knowledge of computers, hardware, Windows, networking, storage devices, operating systems and technology through a series of carefully selected questions designed for enthusiasts, students and IT professionals.

The goal is simple: answer all questions correctly and prove that you have what it takes to become a true Computer Technician.

## Features

* 15 computer-related questions
* Multiple choice answers
* Educational explanations after each answer
* Retro-inspired interface
* Lightweight and fast
* No installation required
* Runs directly in any modern web browser
* MIT License

## Topics Covered

* Hardware
* Memory (RAM and ROM)
* Storage devices
* Windows operating systems
* BIOS and UEFI
* Networking fundamentals
* DNS and DHCP
* File systems
* Internet technologies
* Computer history

## Technologies Used

* HTML5
* CSS3
* JavaScript
